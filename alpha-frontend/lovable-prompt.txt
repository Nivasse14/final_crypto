# PROMPT POUR LOVABLE - ALPHA HUNTER PLATFORM

## VISION GLOBALE

Développe une plateforme d'élite financière appelée "Alpha Hunter" qui permet aux investisseurs d'accéder à des données et stratégies utilisées seulement par le top 0,0000001% des traders et investisseurs institutionnels. Cette plateforme identifie et analyse les portefeuilles crypto les plus performants scrapés sur Dune Analytics, révélant les stratégies cachées qui génèrent des gains exceptionnels. L'objectif est de créer un outil qui transforme un investisseur ordinaire en maître de la génération de richesse à travers une expérience utilisateur ultra-premium, des analyses prédictives avancées et des signaux d'investissement exclusifs.

## ARCHITECTURE TECHNIQUE

Le système est composé de trois parties majeures:
1. Backend de scraping et d'enrichissement (existant)
2. APIs Supabase Edge Functions (déployées)
3. Frontend Next.js ultra-premium (à développer)

## APIS DÉPLOYÉES

### 1. API Cielo - Analyse de Wallet

#### Données Complètes
```typescript

// Retourne toutes les données d'un wallet: portfolio, stats, PnL, behavioral analysis
```

#### Endpoints Spécialisés
```typescript

GET /cielo-api/tokens-pnl/{address} // PnL par token avec filtres
```

#### Analytics Avancées
